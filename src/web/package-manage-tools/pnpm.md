:::

## pnpm优点

1. 快速——npm是同类工具速度的将近两倍
2. 高效——node_modules 中的所有文件均克隆或硬链接自单一存储位置
3. 支持单仓库——pnpm 内置了对单个源码仓库中包含多个软件包的支持
4. 权限严格——pnpm 创建的 node_modules 默认并非扁平结构，因此代码无法对任意软件包进行访问
5. 集中存储
6. 依赖隔离
7. monorepo
