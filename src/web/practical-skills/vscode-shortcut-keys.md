- **合并多行**：`Ctrl + J`
- **代码格式化**：`Shift + Option +F`(Mac)、`Shift + Alt + F`
- **代码折叠**：`Ctrl + Shift + [`
- **复制上下行**：`Shift + Alt 上/下箭头`
- **打开或关闭侧边栏**：`Ctrl + B`
- **导航到特定行**：`Ctrl + G`然后输入行数。
- **转到文件中的符号**：`Ctrl + Shift + O`
- **转到工作区的符号**：`Ctrl + T`
- **删除上一个词**：`Ctrl + Backspace`
- **删除一行**：`Ctrl + X`
- **上方或下方添加光标**：`Ctrl + Alt + 上/箭头`
- **列框选择**：`Shift + Alt + 鼠标选中`
- **命令面板**：`Ctrl + P`
